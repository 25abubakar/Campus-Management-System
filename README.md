🎓 Campus Management System:
______________________________

A web-based Campus Management System built using ASP.NET Core MVC, Entity Framework Core, and SQL Server.
This system helps manage students, teachers, courses, and attendance in an organized and efficient way.

🚀 Features:
_____________

👨‍🎓 Student Management:
Add, edit, delete, and view students
Assign courses to students
Track student records
___________________________________________________________________
👨‍🏫 Teacher Management:
Manage teacher profiles
Assign courses to teachers
___________________________________________________________________
📚 Course Management:
Create and manage courses
Assign courses to students and teachers
🧾 Attendance System
Mark student attendance
View attendance by course/date
Filter attendance records
___________________________________________________________________
🔐 System Features:
Clean MVC architecture
TempData success/error messages
Validation using Data Annotations
Responsive UI with Bootstrap
___________________________________________________________________
🛠️ Tech Stack:
ASP.NET Core MVC
Entity Framework Core
SQL Server
C#
Bootstrap 5
Razor Views
